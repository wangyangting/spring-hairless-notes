# 使用 Spring JDBC 访问数据

## 概述
Spring 中使用 JDBC 来访问数据。

## 特性
1. lombok 插件使用，减少 PO 对象的方法编写
2. 使用 Spring JdbcTemplate 来访问 H2（内存数据库）中的数据

## 代码
下载 IDEA，即可直接右键 **RUN** 了。

> 注意: IDEA 需要安装下 lombok 插件即可。
