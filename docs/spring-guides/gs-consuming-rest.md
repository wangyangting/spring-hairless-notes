# 消费 RESTful 风格的服务
获取 API 中的数据

## 概述
Spring 入门的最简单的教程了 - - 、

访问别人提供的接口，获取返回的 JSON 格式的数据

## 特性
1. lombok 插件使用，减少 DTO, POJO 对象的方法编写
2. 访问别人提供的一个接口，然后把 JSON 格式的数据转换成对象

## 代码
下载 IDEA，即可直接右键 **RUN** 了。

> 注意: IDEA 需要安装下 lombok 插件即可。
