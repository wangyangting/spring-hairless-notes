# 使用 Spring JPA 访问数据

## 概述
Spring 中使用 JPA 来访问数据。

## 知识点
1. lombok 插件使用，减少 PO 对象的方法编写
2. 使用 Spring JPA 来访问 H2（内存数据库）中的数据
